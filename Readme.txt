main.m中有两个例子。

现在只支持时间序列的训练和测试样本，即x, y 是n*1向量；如果要支持其他类型的数据，需要修改配置文件中的dataProcessFunction（这里放的是数据处理m文件，需要自己写），并相应的修改prepareData.m, eddPredict.m文件。

netTrainTest.m实现的是对同一训练数据的多次训练(这里训练样本x是矩阵，每一行是一个样本点，y是向量)，如果只训练一次的话，修改配置文件num_train=1即可。其中也包括了对测试样本的训练，这里训练样本可以是多组，准备数据（prepareData.m）中修改即可。

对于像wh的例子，每次训练的数据不一样的情况，可以将每次训练的数据单独保存，在分别运行netTrainTest即可。主要在数据准备上。
